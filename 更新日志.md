## [0.0.2] - 2026-05-07

### ✨ 核心协议大满贯补齐 (Core & Protocols)
- **VMess & Trojan 核心解密**：底层全自研集成 `parse_vmess` 与 `parse_trojan` 解析器，支持将标准的 VMess（含 Base64 标签/配置提取）与 Trojan 协议无缝转换为高规格 `sing-box` 底层配置，完美实现全部 6 大现代主流网络协议的桥接中转。
- **宽容级智能订阅解析器**：升级一键节点解析系统。支持自动识别并递归解码 Base64 订阅源整块密文，并使用智能正则提取任意夹杂在混排文本中的节点分享链接（隔离文本噪音，微信、邮件、剪贴板混排随手粘随手解）。

### 🖥️ 管理面板与高能“活动连接监控” (Web Panel & Connection Monitor)
- **状态监控与节点选择深度解耦（殿堂级 5 栏 UI）**：将控制看板进行物理重构，拆分为独立的「运行状态」和「节点选择」两大面板。
  - **「运行状态」**：全屏免滚动，聚合实时双通道上传/下载波形图与活跃出口节点。**分流模式显示升级为精致的只读标签（`stat-route-mode`）**，排除干扰，聚焦高保真全局观测。
  - **「节点选择」**：全尺寸独立网格。**将「全局代理/绕过大陆」分流下拉框从状态页移入此处**，使代理选择与出口分流策略深度整合；**全面重构页面滚动机制，引入 App 级锁定视口 Flex 自适应结构，整个页面本身绝不产生垂直滚动条**，所有溢出的节点卡片全部限制在专属卡片网格容器（`.node-cards`）内实现局部优雅滚动，提供极致、纯粹的原生应用质感。
  - 导航重塑为 5 大纯粹看板：「运行状态」、「节点选择」、「配置管理」、「连接监控」、「安全审计」，极具秩序美感。
- **全新活动连接追踪（连接监控看板）**：在侧边栏引入高科技 ECG 律动图标，构建全新「连接监控」仪表盘。直观追踪当前云端网关处理的每一个 TCP/UDP 活跃连接的发起 IP、目标主机、匹配分流规则、出口节点，并实时计量各连接的上传/下载累积流量。**数据表全面升级为带有全网格单元格边框与轻量卡片包装的设计（`table-container`）**。
- **全局品牌色彩系统建立（色彩对比度Bug修复）**：在 CSS `:root` 中完整声明了之前缺失的 `--primary` 和 `--primary-rgb` 主色调变量，从而彻底修复了连接监控标题右侧“活跃连接数角标”在冷启动或无连接时，因背景落回淡色背景、文字呈白色导致的“隐形方框”问题，使其呈现出亮丽、高识别度的深蓝色圆角角标（数字极易辨识）。
- **5s 低功耗静默刷新**：前台激活该看板时自动每 5s 轮询，切回其他页面时自动销毁定时器，零无用功损耗。
- **实时本地毫秒级过滤**：支持通过主机名、目标 IP 或发起 IP 对活动连接进行即时就地模糊搜索，无需额外请求。
- **精准连接切断与一键断开**：支持点击每行最右侧的 X 按钮精准切断特定高负载连接，或使用右上角「断开全部」按钮，强行重置所有长连接。
- **全局直连模式支持**：路由规则新增「全局直连（direct）」档位，重构校验逻辑，一键重载后内核完全不经过任何代理网关，满足临时直连和调试刚需。
- **安全审计日志「Flex弹性自适应」与序号列升级（零整页滚动条）**：
  - **弹性自适应调优**：放弃了硬编码高度或 %vh 阈值限制，将 `#panel-logs` 及其内层包裹卡片（`.card`）重构为严格的 Flex 比例缩放模型（`flex: 1; display: flex; flex-direction: column; overflow: hidden;`），并将日志数据表容器 `.panel-log-table-wrap` 设为 `flex: 1; min-height: 0;`。这使得日志表格高宽比能够**在各种极端屏幕高度下，100% 自动精确撑满并锁定在当前可用视窗内**。外部大卡片高度被严密锁死，**绝对没有任何可能产生页面级/外部垂直滚动条**，实现了极其完美、绝对整洁纯净的零滚动条（Zero-Scrollbar）原生仪表盘质感。
  - **新增序号列与列宽精准校正**：为安全审计日志数据表新增首列「序号」字段，对所有拉取到的操作日志流水按自增顺序居中高对比度排列；同时重构了 CSS 列选择器（`nth-child`），为「序号（65px）」、「时间（170px）」、「IP（110px）」、「用户（80px）」与「操作（120px）」分配了绝对合理、绝不重叠挤压的专属列宽，完美解决了多列挤压重合的显示瑕疵。
  - **列名中文化**：同步将「节点库管理」页面主数据表首列原有的英文占位符号「#」重命名为「序号」，使整套系统的表格中文化体验达成 100% 连贯与统一。
  - **活动连接监控序号化**：为「活动连接监控」主表格也同步新增了首列「序号」列，使当前后台拉取到的所有实时代理连接流水获得最规整、自增的居中序号，大幅提升了对复杂多并发网络连接流进行排查时的点对点追踪效率。
- **全局视口边缘留白优化（局部滚动架构重构）**：彻底重构了页面的滚动架构，将 `body` 设置为严格锁定视口高度（`100dvh`）并阻止窗口级滚动，同时解除页脚的 `fixed` 固定定位使其融入正常流。将核心内容容器 `.main` 升级为独立的自适应滚动区域（`overflow-y: auto`）。这从底层彻底解决了超长内容（如审计日志）滚动到底部时被页脚遮挡或“零边界黏合”的顽疾，确保页面任何卡片滑到底部时，都拥有原生的 `1.5rem`（24px）纯净留白，实现了真正比肩原生客户端（App-like）的局部滚动体验。

### 🔒 工业级安全特性升级 (Security Hardening)
- **不逆派生订阅令牌 (Hashed Sub-Token)**：设计了基于管理员密码（`PANEL_PASSWORD`）与会话私钥（`SESSION_SECRET`）进行 SHA256 加盐单向不可逆派生的 16 位订阅 Token 体系，杜绝公网直接暴露敏感密码或节点数据的风险。
- **多端双重认证分发流**：
  - **网页浏览器下载**：自动识别 Session Cookie 免参数直接下载。
  - **外部第三方客户端同步**：Clash Verge、v2rayN、Shadowrocket 等客户端订阅时，只需在请求后缀携带 `?token=[SubToken]` 即可安全拉取明文代理节点，保障手机端自动秒级同步。

### 🔄 三合一多端订阅导出中心 (3-in-1 Export Hub)
- **Clash 统一 YAML 订阅**：开发 `/api/export/clash`，将当前运行的活跃节点导出为带有负载均衡与规则分流的 Clash 标准 YAML 配置文件（`nethub_clash.yaml`）。
- **V2ray 通用 Base64 订阅**：开发 `/api/export/v2ray`，将节点反向拼接成标准 `vmess://`、`vless://` 链接列表，并用 Base64 加密为标准的 V2ray 格式（`nethub_v2ray.txt`）。
- **sing-box 终端配置订阅**：开发 `/api/export/singbox`，支持直接流式返回原始 `config.json` 供客户端直接订阅（`nethub_config.json`）。

### 💄 极致体验调优与 Bug 修复 (UX & Bug Fixes)
- **彻底斩断前端 IIFE 作用域闭包死锁**：修复了因顶部未声明 `let trafficSSE = null;` 导致 `stopTrafficSSE` 运行时抛出 `ReferenceError: trafficSSE is not defined` 的致命 Bug，解除了控制面板主页首次载入、切签初始化死锁。
- **导出订阅按钮合理重排**：将导出触发按钮从喧宾夺主的顶部 Header 标题栏移除，合理归并在「库与节点」管理区，在“配置导入/管理”右侧配置专门的触发器，并配备了带一键复制链接、一键下载的多功能 Modal 对话框。
- **安全解耦 DOM 异常节点**：清理了前端已废弃 DOM 元素的直接显示与隐藏操作，彻底阻断由于空对象引用导致的 `TypeError` 运行时 JavaScript 抛错。

## [0.0.1] - 2026-05-03

### ✨ 核心功能 (Core)
- **多协议中转**：支持 VLESS (Reality), Hysteria2, TUIC, Shadowsocks 等主流协议桥接为标准 HTTP 代理。
- **内核自动化管理**：内置 `run.py` 启动器，支持自动检测平台并下载/更新 `sing-box` 内核。
- **配置热加载**：检测到 `config.json` 变动时，内核支持秒级自动重启/重载配置。
- **高性能架构**：基于单二进制内核运行，内存占用低，中转效率高。

### 🖥️ 管理面板 (Web Panel)
- **可视化界面**：基于 FastAPI 实现的现代化管理后台，支持移动端自适应。
- **节点管理**：支持查看节点列表、手动切换出口节点、开启/关闭指定节点。
- **订阅导入**：支持从 URL 链接或原始分享链接直接导入节点，并自动持久化。
- **实时审计**：内置日志查看器，可实时监控系统运行状态及连接记录。
- **安全认证**：实现基于 Session 的登录认证系统，支持强密码验证与登录频率限制。

### 🔒 安全特性 (Security)
- **加密节点库 (Vault)**：引入加密存储机制，原始订阅链接加密保存于 `vault.enc`，确保敏感信息不落地。
- **环境变量驱动**：所有敏感密钥（密码、API Secret）通过 `.env` 管理，并默认被 Git 忽略。
- **权限隔离**：Docker 环境下支持非 Root 用户运行，降低系统安全风险。

### 🚀 部署 with Ubuntu (Deployment)
- **Ubuntu 一键部署**：提供 `一键部署.sh` 脚本，自动化安装 Docker 环境并拉起全栈服务。
- **容器化支持**：完整支持 Docker Compose 部署，实现内核与面板的服务解耦。
- **跨平台适配**：完美兼容 Windows 开发环境与 Linux 生产环境。
